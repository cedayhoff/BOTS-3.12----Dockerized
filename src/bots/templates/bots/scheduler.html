About Route Scheduling

The Route Scheduler allows you to automate the execution of EDI routes at specific times and intervals.

Schedule Types

  • Minutes: Run every X minutes
  • Hours: Run every X hours
  • Daily: Run daily at specific time or every X days
  • Weekly: Run weekly on specific day/time or every X weeks
  • Monthly: Run monthly on specific day/time or every X months

Default System Schedules

The following schedules run automatically and cannot be modified:

  • Process New Messages: Every 5 minutes
  • Retry Communications: Every hour
  • Daily Cleanup: Daily at 2:00 AM
